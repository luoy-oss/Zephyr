# Zephyr - 弹琴工具

## v1.0.0 (2026-06-01)
- 初始版本发布
- 基于 Flutter 构建安卓应用
- 支持自定义琴谱导入（txt格式）
- 琴谱管理：导入、搜索、选择、删除
- 播放引擎：可调BPM（30-200）、速度倍率（0.25x-3x）
- 延迟开始：0-15秒倒计时
- 按键时长：50-500ms可调（默认100ms）
- Android 无障碍服务模拟屏幕点击
- 游戏内悬浮窗控制（小球模式 + 展开面板）
- 可视化琴键位置校准（行间距、列间距可调）
- 光遇钢琴布局：3行5列（-1到+1共15个音符）
